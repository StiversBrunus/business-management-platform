# Business Management Platform

Sistema desenvolvido como **Trabalho de Conclusão de Curso (TCC)** do curso Técnico em **Desenvolvimento de Sistemas** no SENAI (2018–2020).

O objetivo do projeto foi desenvolver uma **plataforma completa de gestão**, integrando aplicações **web, API e mobile**, permitindo gerenciamento de dados, integração entre serviços e consumo de APIs.

# 🏗 Arquitetura do sistema

O sistema foi desenvolvido utilizando arquitetura baseada em **API REST**, com separação entre:

Backend (API)
Responsável pela lógica de negócio e acesso ao banco de dados.

Frontend (Web)
Interface web para interação com o sistema.

Mobile
Aplicação mobile desenvolvida em Flutter consumindo a API.

Banco de Dados
MySQL para persistência das informações.

Arquitetura simplificada:

Mobile / Web
↓
REST API (Spring Boot)
↓
MySQL Database

# 🚀 Como executar o projeto

git clone https://github.com/stiversbrunus/business-management-platform

---

## 2 Configurar banco de dados

Criar banco MySQL.

Importar scripts SQL localizados no projeto.

---

## 3 Rodar backend

Executar aplicação Spring Boot.

---

## 4 Rodar frontend

Abrir projeto web ou iniciar aplicação React.

---

## 5 Mobile

Executar aplicação Flutter conectando à API.
